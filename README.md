# TicTacToeHackermode
An app which lets you play TicTacToe in 1-Player mode and 2-player mode.

# Features
The Single player mode consists of 3 difficulty levels - Easy, Medium and Hard, of which Hard mode is unbeatable.
When a game starts, a timer starts which stops once the game ends.
If the game is of Single Player Mode, and player wins, then this time is stored in a SQLite Database and all games won are shown in a different activity, with the mode of the game and time taken to win the game.
In the same activity where data is stored, an additional feature of "Clearing" all the data is also available which resets the data.
On the front screen the app shows session best time.
